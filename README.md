## Getting Started

  * Open the ```Chit Chat.xcworkspace``` in Xcode.
  * Change the Bundle Identifier to your domain.
  * Create a project in Firebase. See this [Link](https://firebase.google.com/docs/ios/setup)
  * Download  ```GoogleService-Info.plist``` file and add to the project.
  * Go to [Firebase Console](https://console.firebase.google.com), select your project.
  * Choose **Authentication** from left menu, select **SIGN-IN METHOD** and enable **Email/Password, Google, Facebook**. Refer these guide for [Google](https://firebase.google.com/docs/auth/ios/google-signin) and [Facebook](https://firebase.google.com/docs/auth/ios/facebook-login) Authentication.
  * Run Chit Chat on your iPhone or iOS Simulator.

## Compatibility

This project is written in Swift 4 and requires Xcode 9.1 to build and run.
